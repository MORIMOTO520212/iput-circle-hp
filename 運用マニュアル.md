## SSHでサーバーへ接続する

```bash
# さくらVPSの場合
ssh ubuntu@153.121.73.181 -i '~/.ssh/<秘密鍵>'
```

## ファイル・ディレクトリパス一覧

| **ファイル・ディレクトリ**       | **Path**                                                        |
| -------------------------------- | --------------------------------------------------------------- |
| Nginx設定ディレクトリ            | /etc/nginx/conf.d/                                              |
| ウェブサイトのドキュメントルート | /usr/share/nginx/iput-one.com                                   |
| WordPressテーマファイルパス      | /usr/share/nginx/iput-one.com/wp-content/themes/iput-circle-hp/ |
| WordPressエラーログ              | /var/log/nginx/wordpress.error                                  |
| php.iniパス                      | /etc/php/8.1/cli/php.ini                                        |

## gitを使って更新する

```bash
# さくらVPS
cd /usr/share/nginx/iput-one.com/wp-content/themes/iput-circle-hp/
sudo git pull origin master-wordpress
```

## Nginx 起動/停止/再起動

```bash
sudo service nginx start
sudo service nginx stop
sudo service nginx restart
```

# WordPress初期設定

## 設定 > 一般設定

管理者メールアドレスは「iputone.staff@gmail.com」にする。
メンバーシップにチェックを入れる。クライアント側からユーザー登録をするためである。

## 設定 > 表示設定

ホームページの表示を固定ページにし、ホームページからトップページを選択する。

## 設定 > パーマリンク

パーマリンク構造からカスタム構造を選択し、「/index.php/%year%/%monthnum%/%day%/%postname%/」を入力する。
